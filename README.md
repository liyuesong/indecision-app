# indecision-app

This is a To Do List website building with [`React`](https://reactjs.org).

## setup

Clone the repository

```
cd indecision-app
```

1. Install dependencies with [`yarn`](https://www.npmjs.com/package/yarn) do:
    ```
    yarn install
    ```
   
2. Build.
- Do:
    ```
    yarn run dev-server
    ```
    or,
    using live-server, do:
    ```
    yarn run build
    yarn run serve
    ```
